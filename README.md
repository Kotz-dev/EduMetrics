Desenvolvido por **Valter Junior** para 
ajudar calouros a gerenciar sua vida acadêmica.

## ⚙️ Funcionalidades
- ✅ Controle de notas
- ✅ Frequências e presenças  
- ✅ Cálculo de médias
- ✅ Dark/Light mode
- ✅ Suporte a idiomas

## 🚀 Como usar
### 1. Adicionar matéria
- Clique no botão **+** para adicionar uma nova matéria
- Preencha os dados: nome, aulas previstas, ministradas, presenças, N1 e N2
- O app calcula automaticamente a **média** e o **resultado**

### 2. Remover matéria
- Selecione a matéria na tabela
- Clique no botão **X** para remover

### 3. Salvar dados
- **Arquivo → Salvar** para salvar o arquivo atual
- **Arquivo → Salvar como** para salvar em novo arquivo
- **Arquivo → Abrir** para carregar um arquivo salvo

### 4. Preferências
- **Ferramentas → Sistema de Preferências**
- Escolha o **tema** (Claro/Escuro)
- Escolha a **fonte**
- Escolha o **idioma**
- Clique em **Salvar** para aplicar
